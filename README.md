# 🌐 Personal Portfolio Website

This is my personal portfolio website built to showcase my projects, skills, education, and experience as a Software Engineering student. It serves as a central place where recruiters and collaborators can learn more about my work and technical abilities.

---

## 🚀 Live Demo

👉 https://thatomakhusha.github.io/personalportfolio/

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript (Vanilla)  

---

## ✨ Features

- Fully responsive design for mobile and desktop devices  
- Smooth scrolling navigation between sections  
- Interactive UI elements for better user experience  
- Clean and modern layout to highlight projects and skills  
- Easy-to-read structure for recruiters and visitors  

---

## 🎯 Purpose of the Project

The goal of this project was to:

- Build a professional online presence  
- Practice frontend development skills  
- Showcase my software engineering journey and projects  
- Create a platform for internship and job applications

Through building this project, I improved my skills in:

- Structuring responsive web pages  
- Writing clean HTML, CSS, and JavaScript code  
- Improving UI/UX design principles  
- Deploying websites using GitHub Pages  
- Organising content in a professional developer portfolio format
